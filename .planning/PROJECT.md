# Tenor Creative Redesign

## What This Is
A complete design and content overhaul of the Tenor Creative agency website. It serves small businesses needing essential digital presence (Brochure Websites, Local SEO, GHL CRM setup), acting as an accessible entry point with clear pathways to the parent full-service agency (Prickly Pear Marketing Co.) for advanced needs.

## Core Value
A professional, data-backed, high-conversion digital storefront that simplifies marketing tech for small businesses while seamlessly integrating with the GoHighLevel ecosystem.

## Requirements

### Validated
(None yet — redesigning from scratch)

### Active
- [ ] **Design Overhaul**: Minimalist, modern, and professional aesthetic ("ui-ux-pro-max").
- [ ] **Content Strategy**: Re-structured service offerings based on industry best practices (replacing/validating "4 Rs").
- [ ] **GHL Integration**: Custom UI forms connecting to GoHighLevel API (replacing HubSpot).
- [ ] **Booking Flow**: Integration with existing GHL Calendars.
- [ ] **Brand Bridge**: Contextual upsells and clear pathways to "Prickly Pear Marketing Co." for full-service needs.
- [ ] **Tech Stack**: Retain Nuxt.js + Tailwind CSS foundation.

### Out of Scope
- [ ] **HubSpot Integration**: Removing entirely.
- [ ] **Full Service Offerings**: Video production, CTV, Paid Ads (these belong to Prickly Pear, only referenced as upsells).

## Context
- **Current State**: Functional but visually outdated Nuxt site. Uses "4 Rs" framework which may be too complex/niche.
- **Ecosystem**: Moving fully to GoHighLevel (GHL) for CRM/Booking/Forms.
- **Relationship**: Tenor Creative = "Essentials" (Web, Local SEO, CRM). Prickly Pear = "Full Service" (Ads, Video, Strategy).
- **Design**: User specifically requested "ui-ux-pro-max" quality — minimalist, modern, high polish.

## Constraints
- **Tech**: Nuxt.js, Tailwind CSS.
- **Integration**: Must use GHL API for forms (not just generic embeds where possible, or styled embeds).
- **Timeline**: Redesign takes priority over content migration if content is being rewritten.
