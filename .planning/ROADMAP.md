# Roadmap: Tenor Creative Redesign

**Overview:**
A strategic overhaul of Tenor Creative's digital presence, transitioning from generalist positioning to a focused "Essentials" provider. The roadmap prioritizes establishing the new "Packages" structure and GHL connectivity first, followed by high-fidelity design implementation, and finally deep tool integration and polish.

## Phases

### Phase 1: Foundation & Strategy
**Goal:** Essential structure, global elements, and GHL connectivity are established, while ensuring a clean slate by remediating technical debt.
**Dependencies:** None
**Requirements:** `SERV-04`, `CORE-04`, `BRAND-02`, `TECH-01`
**Success Criteria:**
1. User can view the defined "Essentials" packages and pricing structure.
2. User sees the "Part of Prickly Pear" brand bridge in the global footer.
3. User can access required Legal pages (Privacy/Terms).
4. GHL API client is configured and successfully authenticated.
5. Critical technical debt and known bugs are resolved, establishing a stable codebase.

### Phase 2: Core Experience
**Goal:** Visitors encounter a "ui-ux-pro-max" brand experience on key landing and service pages.
**Dependencies:** Phase 1
**Requirements:** `CORE-01`, `CORE-02`, `SERV-01`, `SERV-02`, `SERV-03`, `BRAND-01`
**Success Criteria:**
1. User receives immediate value proposition via StoryBrand hero on Home page.
2. User can navigate distinct "Brochure Web", "Local SEO", and "CRM" service pages.
3. User understands the "Guide" relationship via the About page.
4. User sees "Need More?" upsell pathways on service pages for advanced needs.

### Phase 3: Conversion & Tools
**Goal:** Visitors can contact, book, and audit their business via fully functional GHL integrations.
**Dependencies:** Phase 2
**Requirements:** `CORE-03`, `GHL-01`, `GHL-02`, `GHL-03`, `TOOL-01`, `TOOL-02`
**Success Criteria:**
1. User can submit a custom-styled contact form that posts directly to GHL CRM.
2. User can book a strategy call via embedded GHL Calendar.
3. User can engage with the site via GHL Chat Widget.
4. User can complete Audit and Onboarding surveys with successful data capture.

### Phase 4: Polish & Launch
**Goal:** The site is credibility-proofed, validated for SEO, and ready for public traffic.
**Dependencies:** Phase 3
**Requirements:** `SERV-05`
**Success Criteria:**
1. User can view relevant case studies/examples on the Portfolio page.
2. Old site URLs successfully redirect to new structure (SEO preservation).
3. Mobile experience meets "ui-ux-pro-max" standards across all pages.
