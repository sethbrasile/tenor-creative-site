# Requirements: Tenor Creative Redesign

**Defined:** Sun Jan 25 2026
**Core Value:** Professional, simple marketing tech for small businesses (Web + SEO + CRM).

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Core Pages
- [ ] **CORE-01**: **Home Page** — High-converting hero, "Digital Storefront" value prop, StoryBrand structure.
- [ ] **CORE-02**: **About Page** — "The Guide" positioning, team intro, trust building.
- [ ] **CORE-03**: **Contact Page** — Primary conversion point with Map and GHL form.
- [x] **CORE-04**: **Legal Pages** — Privacy Policy and Terms of Use (Migrated).

### Services & Sales
- [ ] **SERV-01**: **Web Design Page** — Focus on "Brochure" and "Essentials" websites.
- [ ] **SERV-02**: **Local SEO Page** — Focus on Google Business Profile (GBP) and local visibility.
- [ ] **SERV-03**: **CRM/GHL Page** — Focus on Automation, Unified Inbox, Missed Call Text Back.
- [x] **SERV-04**: **Packages/Pricing Page** — Tiered "Essentials" bundles (e.g., Starter, Growth).
- [ ] **SERV-05**: **Portfolio Page** — Migration of "Sample Sites" to showcase work.

### GHL Integrations
- [ ] **GHL-01**: **Custom UI Contact Form** — Custom Vue form posting to GHL API (replacing HubSpot).
- [ ] **GHL-02**: **Booking Calendar** — Integration of GHL Booking Calendar (Embed or Link).
- [ ] **GHL-03**: **Chat Widget** — GHL Chat Widget installed on all pages.

### Tools & Legacy
- [ ] **TOOL-01**: **Audit Survey** — Migration/Redesign of existing audit survey tool.
- [ ] **TOOL-02**: **Onboarding Survey** — Migration/Redesign of existing onboarding survey.

### Brand Bridge
- [ ] **BRAND-01**: **"Need More?" Section** — Clear upsell path to Prickly Pear for advanced services (Ads, Video).
- [x] **BRAND-02**: **Prickly Pear Footer** — "Part of Prickly Pear Marketing Co." link/badge.

### Technical Foundation
- [x] **TECH-01**: **Remediation** — Address existing technical debt and fix known bugs in the codebase.

## v2 Requirements

## Out of Scope
Explicitly excluded.

| Feature | Reason |
|---------|--------|
| **HubSpot** | Replacing completely with GHL. |
| **The 4 Rs** | Replacing framework with standard Service/Package structure. |
| **Paid Ads / Video** | Belong to Prickly Pear (parent agency). |
| **E-commerce** | Not part of "Essentials" offering. |
